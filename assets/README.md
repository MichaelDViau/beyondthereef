# Assets

- `common/logo.png`: site logo + favicon.
- `indexslider/`: images used in the homepage hero slider.
- `tour-sliders/<tour-page>/`: dedicated folder for each tour page photo slider.

For each tour slider folder, add files named:
- `slide-01.jpg`
- `slide-02.jpg`
- `slide-03.jpg`
- `slide-04.jpg`

The tour pages are already linked to these paths.
